Build a particle diagram, in four moves

  1. Name each size. Type a label for the small, medium, and large particles — element symbols or charges like Na⁺ and Cl⁻. Use the ⁺ ⁻ ² ³ keys for superscript charges.
  2. Set the counts. Bump each size up with the + steppers, then drag the particles into the arrangement you want.
  3. Go metallic. Flip on Sea of e⁻ to shade the field and scatter delocalized electrons — the picture of metallic bonding.
  4. Copy it. Hit Copy image and paste your diagram straight into a doc, slide, or quiz.

Neat freak? Flip on Snap to grid to click particles into a tidy lattice — perfect for ionic crystals and close-packed metals. Set Background → Clear to copy a see-through diagram. Snapping only tidies placement; it never decides how many particles you use.
